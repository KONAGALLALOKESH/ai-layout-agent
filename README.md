# AI Layout Agent

A chat-based layout transformation tool built using React and Node.js.

Users can give natural language instructions like:

- Convert to 9:16
- Move headline to top
- Make headline smaller
- Move offer badge higher
- Keep product large

The application updates the layout JSON and wireframe preview in real time.

---

# Tech Stack

## Frontend
- React
- Vite
- Axios

## Backend
- Node.js
- Express.js

# Setup Instructions

## Clone Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
```

---

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```

---

## Backend Setup

```bash
cd server
npm install
node index.js
```

Backend runs on:

```text
http://localhost:3001
```
